# therossfisher.xyz — Resume Frontend

Static resume site hosted on AWS S3 + CloudFront with HTTPS.

## Live Site
[therossfisher.xyz](https://therossfisher.xyz)

## Stack
- **Hosting:** AWS S3 (static website)
- **CDN/HTTPS:** AWS CloudFront
- **DNS:** AWS Route53
- **Infrastructure:** Terraform
- **CI/CD:** GitHub Actions

## Deployment
Push to main triggers automatic deployment via GitHub Actions.
